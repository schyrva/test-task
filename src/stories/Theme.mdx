## Spacing

Our spacing system is built on a 4px base unit (0.25rem in Tailwind), creating a consistent rhythm throughout the interface.

## Elevation

We use a combination of shadows and z-index values to create depth:

- **None**: No shadow
- **SM**: Small shadow for subtle elevation
- **MD**: Medium shadow for cards and interactive elements
- **LG**: Large shadow for modals and popups
- **XL**: Extra large shadow for elements that need significant elevation

## Border Radius

We use a consistent set of border radius values to soften edges:

- **None**: 0
- **SM**: 0.125rem (2px)
- **Default**: 0.25rem (4px)
- **MD**: 0.375rem (6px)
- **LG**: 0.5rem (8px)
- **XL**: 0.75rem (12px)
- **2XL**: 1rem (16px)
- **Full**: 9999px (for pills and circular elements)
